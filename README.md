🎬 StreamVerse

A Production-Ready Netflix-Inspired Streaming Platform built with React, TypeScript, Firebase, TMDB API, and Modern Web Technologies.

🚀 Overview

StreamVerse is a modern streaming platform inspired by the user experience of popular entertainment applications. It delivers a seamless movie and TV show discovery experience with real-time content powered by TMDB, secure authentication through Firebase, dynamic trailer playback, responsive design, and a premium user interface.

The project was built to demonstrate production-level frontend development practices, scalable architecture, API integration, authentication workflows, performance optimization, and modern UI/UX implementation.
✨ Features

- 🔐 Firebase Authentication
- 🎥 TMDB Movie & TV Show Discovery
- 🔍 Real-Time Search
- ▶️ Trailer Playback
- ❤️ Watchlist Management
- 📱 Fully Responsive Design
- ⚡ Fast & Optimized Performance
- 🎨 Modern Netflix-Style UI

## 🛠️ Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Firebase
- TMDB API
- Vercel
